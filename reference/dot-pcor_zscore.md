# Compute the z-score of a partial correlation, see https://github.com/cran/SILGGM/blob/master/src/SILGGMCpp.cpp

Compute the z-score of a partial correlation, see
https://github.com/cran/SILGGM/blob/master/src/SILGGMCpp.cpp

## Usage

``` r
.pcor_zscore(pcor, n)
```

## Arguments

- pcor:

  the partial correlation

- n:

  the number of samples

## Value

the z-score of the corresponding partial correlation
