# Function to compute Euclidean distance with "pairwise.complete.obs" method

Function to compute Euclidean distance with "pairwise.complete.obs"
method

## Usage

``` r
.euclidean_pairwise_complete_obs(v1, v2)
```

## Arguments

- v1:

  the first vector

- v2:

  the second vector

## Value

the Euclidean distance
