# Perform walk-trap community detection algorithm on a network

Perform walk-trap community detection algorithm on a network

## Usage

``` r
.walktrap(igs, vertex_symbol = "name", steps = 10)
```

## Arguments

- igs:

  a list containing an igraph object named "ig"

- vertex_symbol:

  the node attribute that represents the symbol of the node

- steps:

  the number of steps in walk-trap algorithm

## Value

a list object
