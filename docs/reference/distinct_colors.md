# Distinct colors for large categorical datasets

Colors derived from the R package colorspace colorspace: A Toolbox for
Manipulating and Assessing Colors and Palettes
https://cran.r-project.org/web/packages/colorspace/index.html License:
BSD_3_clause +
https://cran.r-project.org/web/packages/colorspace/LICENSE

## Usage

``` r
distinct_colors(n)
```

## Arguments

- n:

  Number of colors desired

## Value

Color palette
