# Function to count triangle orbits

Function to count triangle orbits

## Usage

``` r
.count_tri_orbits(v, u, w, edge_list, GOM)
```

## Arguments

- v:

  a vertex id

- u:

  a vertex id

- w:

  a vertex id

- edge_list:

  a matrix with 3 columns, the third column = "sign"

- GOM:

  the graphlet orbit matrix

## Value

the updated graphlet orbit matrix
