# Perform Louvain community detection algorithm on a network

Perform Louvain community detection algorithm on a network

## Usage

``` r
.louvain(igs, vertex_symbol = "name", resolution = 1)
```

## Arguments

- igs:

  a list containing an igraph object named "ig"

- vertex_symbol:

  the node attribute that represents the symbol of the node

- resolution:

  resolution in Louvain algorithm

## Value

a list object
