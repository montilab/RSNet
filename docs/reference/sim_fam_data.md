# Simulate family-based correlated data

Simulate family-based correlated data

## Usage

``` r
sim_fam_data(num_families = 100, members_per_family = 5, h_sq = 0.5, Sigma)
```

## Arguments

- num_families:

  number of family

- members_per_family:

  number of individuals in each family

- h_sq:

  inheritability

- Sigma:

  the covariance matrix of variables

## Value

an ExpressionSet object
