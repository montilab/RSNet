# Package index

## Network Structure Inference Methods

- [`ensemble_ggm()`](https://montilab.github.io/RSNet/reference/ensemble_ggm.md)
  : Resampling-based Markov Network
- [`ensemble_cgbn()`](https://montilab.github.io/RSNet/reference/ensemble_cgbn.md)
  : Resampling-based conditional Gaussian Bayesian Network
- [`consensus_net_ggm()`](https://montilab.github.io/RSNet/reference/consensus_net_ggm.md)
  : Create consensus network of resampling-based Markov network
- [`consensus_net_cgbn()`](https://montilab.github.io/RSNet/reference/consensus_net_cgbn.md)
  : Create consensus network of resampling-based conditional Gaussian
  Bayesian network
- [`resample()`](https://montilab.github.io/RSNet/reference/resample.md)
  : Perform data resampling
- [`resample_cluster()`](https://montilab.github.io/RSNet/reference/resample_cluster.md)
  : Perform data resampling from correlated data
- [`matrix_p_adjust()`](https://montilab.github.io/RSNet/reference/matrix_p_adjust.md)
  : Perform adjustments of p-values on a p x p matrix
- [`upper_tri_to_matrix()`](https://montilab.github.io/RSNet/reference/upper_tri_to_matrix.md)
  : Reconstruct the symmetric matrix from upper triangular vector

## Graphlet-based Methods

- [`gdvm_gcm()`](https://montilab.github.io/RSNet/reference/gdvm_gcm.md)
  : Find graphlet degree vector (GDV) of each vertex and graphlet
  correlation matrix of the given network
- [`signed_gdvm_gcm()`](https://montilab.github.io/RSNet/reference/signed_gdvm_gcm.md)
  : Function to count signed graphlets
- [`gdv_distance()`](https://montilab.github.io/RSNet/reference/gdv_distance.md)
  : Find the graphlet degree distance between two vertices
- [`intra_gdv_distance()`](https://montilab.github.io/RSNet/reference/intra_gdv_distance.md)
  : Pair-wise GDV distance
- [`paired_gdv_distance()`](https://montilab.github.io/RSNet/reference/paired_gdv_distance.md)
  : Compute Paired Graphlet Degree Vector (GDV) Distances
- [`gcm_distance()`](https://montilab.github.io/RSNet/reference/gcm_distance.md)
  : Distance between two Graphlet Correlation Matrices

## Community Detection Methods

- [`community_detection()`](https://montilab.github.io/RSNet/reference/community_detection.md)
  : Perform community detection on a network
- [`remove_isolated()`](https://montilab.github.io/RSNet/reference/remove_isolated.md)
  : Remove isolated nodes from a network
- [`remove_small_community()`](https://montilab.github.io/RSNet/reference/remove_small_community.md)
  : Remove small communities in a network

## Centrality Methods

- [`centrality()`](https://montilab.github.io/RSNet/reference/centrality.md)
  : Compute essential centrality of a network

## Differential Connectivity Methods

- [`null_ggm()`](https://montilab.github.io/RSNet/reference/null_ggm.md)
  : Differential connectivity nulls via permutation/bootstrap + GGM
- [`permutation_diffanal()`](https://montilab.github.io/RSNet/reference/permutation_diffanal.md)
  : Permutation for differential connectivity analysis
- [`bootstrap_diffanal()`](https://montilab.github.io/RSNet/reference/bootstrap_diffanal.md)
  : Bootstrap for differential connectivity analysis (pooled bootstrap)
- [`diff_centrality()`](https://montilab.github.io/RSNet/reference/diff_centrality.md)

- [`diff_gdv()`](https://montilab.github.io/RSNet/reference/diff_gdv.md)
  : GDV distance-based Differential Connectivity Analysis

- [`model_mpa()`](https://montilab.github.io/RSNet/reference/model_mpa.md)
  : Simulate network via modular preferential attachment model
- [`model_sbm()`](https://montilab.github.io/RSNet/reference/model_sbm.md)
