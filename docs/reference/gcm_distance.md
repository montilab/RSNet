# Distance between two Graphlet Correlation Matrices

Distance between two Graphlet Correlation Matrices

## Usage

``` r
gcm_distance(gcm1, gcm2)
```

## Arguments

- gcm1:

  the GCM of a network

- gcm2:

  the GCM of another network

## Value

numeric; the Euclidean distance between the upper triangular of gcm1 and
gcm2
