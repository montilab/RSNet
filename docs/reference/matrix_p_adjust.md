# Perform adjustments of p-values on a p x p matrix

Perform adjustments of p-values on a p x p matrix

## Usage

``` r
matrix_p_adjust(mx_p)
```

## Arguments

- mx_p:

  a p x p matrix, (i,j) represents the p-value of the partial
  correlation between node i and node j

## Value

a p x p matrix, (i,j) represents the adjusted p-value of the partial
correlation between node i and node j
